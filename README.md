# Uso de IdP en Python

Ejercicio realizado en clase para demostrar el funcionamiento de un IdP.

## ¿Qué es un IdP?

**Identity Provider (IdP)** es un servicio que almacena y verifica las identidades de los usuarios, su uso suele estar relacionado con proveedores de **Single Sign-On (SSO)**. [Leer más...](https://www.cloudflare.com/es-es/learning/access-management/what-is-an-identity-provider/)

## ¿Qué es Auth0?

Es una plataforma que provee gestión de identidades y accesos a través del uso de **OAuth**. [Leer más...](https://identitymanagementinstitute.org/difference-between-oauth-and-auth0/)

## ¿Qué es OAuth (Open Authorization)?

Es un protocolo estándar que permite implementar seguridad y controlar el acceso a los datos de los usuarios o recursos. Este protocolo es usado como un marco de trabajo para la autorización.
